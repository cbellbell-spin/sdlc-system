description: Show the full portfolio status — all initiatives with phase, named PM, and status. Use at the start of a session or to get a portfolio overview.
---

Read `_portfolio_registry.md` from the root of the shared folder.

If the file does not exist: create it with the header row only, note the portfolio is empty, and ask the PM whether to start a new initiative or triage an incoming idea.

Display the portfolio as a clean table:

| Initiative | Named PM | Phase | Status | Last Updated |
|-----------|----------|-------|--------|--------------|

After displaying:
- Surface any initiatives with status "At gate" — these are awaiting a leadership decision
- Flag any initiatives where Last Updated is more than 30 days ago and status is Active
- Ask: "Which initiative do you want to work on, or would you like to start a new one?"
